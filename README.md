In this delegation lab I practised event delegation with different methods.  The first exercise gave practice on event delagation and closest() method.  It showed how attaching one listener uses the bubble up approach to find out what was clicked instead of attaching many event listeners.  The second exercise gave practice on a dynamic container that uses matches() to target what was removed from the container.  The parent container used delegation to find whatever child or children that was added to it.  The third exercise gave practice on event delegation using closet() and matches() methods.  It also used the data-filer attribute which in turn uses the dataset.filter to find out what was selected.  The last exercise gave practice on how event delegation handles double-clicks.  It showed how tags and css styles can be added by using delegation with closet() and matches().
